# 「本地」项目简介
**在地人文体验预订平台 · 2026年3月 · 内部工作版**

---

这份文件写给对这件事感兴趣、想要更完整了解它的人——无论是在考虑加入团队、参与运营，还是只是想搞清楚我们在做什么。这是目前我们最诚实的一份答案。

它不是一份融资材料，而是一份工作文件：我们看到了什么问题、我们打算怎么做、我们在哪里还不确定。

---

## 一、我们在做什么

「本地」是一个连接旅行者与在地文化体验的预订平台。我们把那些只有本地人才懂的日常——手艺、空间、故事、技艺——转化为每个旅行者都能预订、参与、带走记忆的体验产品。

起步城市：大同（山西）。原因在后面展开。

平台连接两端：

- **用户端**：对"进入真实在地日常"有需求的旅行者，但找不到可信任的入口
- **供给端**：有独特内容的本地人或机构，但没有工具把它变成可持续的体验产品

---

## 二、问题：我们在解决什么

### 需求端：旅行者想要"真实"，但找不到入口

越来越多的旅行者不满足于景点打卡。他们想进入真正的日常——当地人去哪里吃早饭、周末做什么、哪条街只有本地人才走。现有渠道给不了这些：

- OTA 卖门票和标准行程，无法承载非标体验
- 内容平台（小红书、抖音）能提供灵感，但没有成交和履约闭环
- 民宿能住进去，但没有人带你真正进入这座城市的日常

用户访谈（10位）的核心发现：

> "不会参加纯观察式的。觉得'做'比'看'更吸引我——要真的动手，或者有真实的对话，才会觉得这趟值了。"
> ——访谈用户，25–35岁，互联网从业者

- **参与式体验的心理定价是观察式的3–5倍**
- 当体验绑定了真实专家资质（研究员、修复师），愿意支付"景点门票的几倍"
- 理想路径：看到即预订——但目前这条路在市场上根本不存在

### 供给端：内容有，但没有预订闭环

这不是大同独有的问题，是整个非标文化体验市场的结构性特征。我们对65个非遗体验工坊（大漆、剪纸、金缮等品类）做了系统调研：

> **65个工坊中，100% 的成交路径是：**
> 小红书帖子 → 私信询问 → 加微信 → 确认报名 → 转账
>
> 没有任何一家做到"点击即订"。

其中点赞最高的一条内容有6,979个赞——但背后仍然是私信成交。内容热度不等于交易效率。

这套摩擦链路的后果：

- 有意愿的用户大量流失在"问了没回、回了嫌麻烦"的环节
- 体验提供方把精力消耗在售前沟通上，无法专注内容本身
- 出了问题（爽约、质量不达预期），双方都没有任何保障机制

## 三、为什么是现在

三个叠加的时机：

### ① 文化事件激活了流量，但深度体验供给几乎为零

《黑神话：悟空》27个取景地落在山西，大同、云冈成为全国级热搜目的地：

- 大同旅游搜索量增幅 +20%
- 山西景区门票上线两个月收入 ¥1.6亿
- 华严寺游客较上年同期增幅 +50%

游客到了，但除了景点门票，几乎没有更多可以"买"的在地体验。这是一个需求被激活、供给严重滞后的窗口。

### ② 旅行行为的结构性转变

"在地体验"、"本地人带你玩"在小红书搜索量连续三年双位数增长；非遗工坊、城市漫步在内容平台上有真实付费转化。65条大漆工坊记录中，头部工坊单帖评论超200条，大部分是真实报名询问。这不是趋势预测，是已经发生的行为。

### ③ 北方文化遗产市场是真实地理空白

现有非遗体验工坊99%集中在成都、上海、武汉等南方城市。山西仅有极少量平遥推光漆器记录（且为另一技法），大同完全没有任何非遗体验工坊记录。这不是"竞争还不激烈"——是竞争尚未开始。

## 四、竞品格局：为什么没人先做

| 竞品 | 定位 | 北方文化遗产覆盖 | 核心结构性问题 |
|------|------|----------------|--------------|
| OTC 小镇 | 南方慢生活体验（大理、泉州等） | 无 | 产品美学是"自然·慢生活·治愈系"，与北方文化遗产的内容密度和叙事语言根本不匹配 |
| 携程玩乐 | 头部OTA体验板块 | 有景点票，无深度非标 | SKU 必须标准化；抽佣30%+，Host 不愿进 |
| 美团玩乐 | 本地生活体验 | 以低价团购为主 | 价格竞争逻辑，无法托举高品质小众体验 |
| 小红书（私域成交） | 内容种草+私信成交 | 有内容覆盖，无成交闭环 | 不做供给侧筛选；正在封堵私域交易 |
| Airbnb Experiences | 全球体验预订 | 2022年完全退出中国 | 文化错配、信任障碍、无微信生态 |

**为什么 OTC 没有北上？**

OTC 的产品方法论是"自然·慢生活·治愈系"——农场采摘、花饼制作、Gin 酒工坊。这套叙事语言在云南大理极为自洽，却和大同的壁画修复、西安的器物美学、洛阳的古建叙事根本不匹配。北方文化遗产体验需要更高的内容密度和不同的 Host 孵化逻辑。这是认知壁垒，不只是资源壁垒——而这正是我们相信自己有优势的地方。

**小红书/抖音加速交易层布局怎么办？**

横向内容平台不做供给侧筛选和产品化。即便平台建好了交易管道，它无法告诉用户哪个向导经过品质认证、无法处理退款投诉。从65个工坊样本来看，即使内容热度极高，私信成交仍然是唯一路径——内容平台没有动力去解决这个结构性问题，因为那不是它的商业模型。平台封堵私域交易，对我们反而是利好。

## 五、产品：我们卖什么

### 好的体验产品的四个判断标准

1. **叙事先行**：每个体验必须能回答"我为什么在这里、这件事为什么重要"。文化解释权是最核心的差异化，也是定价权的基础。
2. **主动参与**：观看不是体验，动手或真实对话才是。参与式体验的心理定价是观察式的3–5倍。
3. **在地不可替代**：这个体验只能在这里、只能由这个人完成——稀缺性是定价权的来源。
4. **有东西可以带走**：实物带走物是付费意愿的催化剂（"花钱换到了一件东西"），也是自发传播的物理载体。大漆工坊市场里，"带走一把漆扇"是最重要的产品成功因素之一。

### 大同产品方向（设想，Host 均待确认）

| 角色 | 品类 / 代表体验 | 潜在 Host 方向 | 参考客单价 |
|------|--------------|--------------|----------|
| **旗舰锚点** | 云冈壁画修复工作坊——跟修复师了解矿物颜料，亲手临摹，带走编号临摹卡 | 云冈石窟研究院文保部门（尚待接触确认） | ¥398–498 |
| **高频主粮** | 城市漫步——由在大同长大的年轻人带队，不去景点，去胡同和本地日常 | 本地文化向导（待开发） | ¥168–228 |
| **高频主粮** | 东南邑空间叙事——走主理人才懂的街区改造史和选址逻辑 | 东南邑艺术街区相关人员（尚待接触确认） | ¥188–268 |
| **高频主粮** | 广灵剪纸轻手作——做一件非遗，带走作品，适合高频排期 | 广灵剪纸传承人（待开发） | ¥168–228 |
| **高频主粮** | 大同早市食物——以"买菜"为线索走一遍大同早市，了解只在这里有的食材 | 本地民宿主、老字号食铺（待开发） | ¥148–198 |

*以上 Host 方向均为基于市场判断的设想，任何一方的合作意向均尚待逐一接触确认，不能视为既有资源。*

### 带走物设计思路：壁画颜料临摹卡

参考大漆工坊"带走一把漆扇"的逻辑，云冈旗舰体验的带走物设计思路：体验者用真实矿物颜料（赭石、石青、石绿——与云冈壁画同材质），在专用素纸上临摹一小块云冈典型纹样，装进印有窟号和日期的小木盒带走。

- 材料成本约¥15–30；每一张带编号，独一无二
- 天然适合小红书"我在云冈做了一件事"内容传播
- 任何人可以开漆扇工坊；但用云冈的纹样和真实矿物颜料做这件事，只能在这里发生

## 六、收入模型：三引擎结构

### 为什么单一抽佣模型不够用

¥298 × 25% × 5人/场 = ¥373/场。要达到月平台收入¥5万，需要约671单——对单城早期文化体验平台，这几乎不现实。问题不在抽佣比例，而在**每场可捕获价值太低、收入来源过于单一**。解法是三台引擎分工协作。

### 引擎 A：云冈旗舰公开场（品牌锚点，不走量）

每场4–6人，限额排期，固定时段，带编号带走物。月目标8–16场。

核心逻辑：靠它建立品牌可信度，而不是靠它产生大量收入。没有这个锚点，城市漫步就是普通旅游产品；有了它，所有 SKU 都在"我们是做云冈研究院壁画修复体验的平台"这个背书下销售。

> **单场测算**：6人 × ¥498 × 28% ≈ **¥836** 平台毛收入

### 引擎 B：非云冈高频 SKU（上座率主粮，真正养活平台）

城市漫步、东南邑空间叙事、广灵剪纸、食物市集。每场6–12人，工作日+周末均可排期，平台对 Host 的控制权更强，上座率更可预测。月目标40–60场。

云冈的稀缺性决定了它只能做贵不能做多；这些 SKU 的可复制性决定了它们能高频出团，成为收入主干。

> **城市漫步示例**：10人 × ¥198 × 25% = **¥495**/场

### 引擎 C：加购与衍生收入（把薄抽佣变厚）

编号带走物（壁画颜料临摹卡装框版）、联名讲义/证书、系列票/双场通票。目标转化率15–20%参与者，平台毛利¥50–80/件。

在体验本身议价空间有限的情况下，加购是最自然的利润扩展方式，毛利率显著高于抽佣。

### M1–M4 大同单城收入爬坡（假设性测算）

| 时间节点 | 月场次 | Host 数 | 平台月收入（估） | 核心任务 |
|---------|--------|--------|--------------|---------|
| **M1** | ~30场 | 3–4 | ~¥1.2万 | 校准月：哪个SKU最容易坐满？哪个Host最稳？ |
| **M2** | ~48场 | 5–6 | ~¥2.6万 | 扩张：城市漫步双路线，剪纸正式产品化 |
| **M3** | ~65场 | 6–7 | ~¥4.0万 | 固化：排期结构清晰，加购率开始提升 |
| **M4** | ~76场 | 6–8 | ~¥5.0万 | 验证：单城模型跑通，具备复制至第二城市的前提 |

*所有数字均为假设性测算，需以实际排期、成团率、Host 稳定性数据为准进行校正。M1 的核心目标不是收入，而是把SKU、Host、上座率逐步校准。*

**两个 Phase 1 运营纪律：**
1. 每个 Host 只配一个主 SKU——避免脚本和品控同时失控
2. 工作日做频次（城市漫步/剪纸），周末做利润（云冈旗舰/空间叙事）

## 七、供给侧策略：Host 从哪来

### Phase 1 优先接触方向

以下是基于市场判断认为值得优先接触的方向。需要明确说明：以下所有方向均尚未经过正式接触，合作可能性完全待确认，不构成任何承诺。

| Host 方向 | 体验类型 | 当前状态 |
|----------|---------|---------|
| 云冈石窟研究院文保部门 | 壁画修复工作坊（旗舰锚点） | 尚待正式接触，是否有合作可能性未知 |
| 大同本地文化向导 | 城市漫步 | 待开发，无现成联系人 |
| 东南邑艺术街区相关人员 | 空间叙事 | 尚待正式接触，意向未知 |
| 广灵剪纸传承人 | 轻手作非遗 | 待开发 |
| 平城学堂相关人员 | 人文沙龙/城市课堂 | 尚待正式接触，意向未知 |

**特别说明**：云冈石窟研究院的合作是整个旗舰产品线的前提。如果这条线无法推进，旗舰锚点就需要重新设计，进而影响整体品牌定位。这是 Phase 1 最需要尽快验证的单一关键变量。

### Host 孵化逻辑

核心工作不是招募，而是帮助潜在 Host 发现并描述自己体验的价值，降低他们的启动门槛：

1. **深度访谈（1–2次）**：挖掘最独特的知识/技能/日常场景；确认运营意愿和时间上限
2. **产品化协作**：团队协助撰写产品文案、设计体验流程脚本（开场→核心→高光时刻→带走物→结尾）、确定定价
3. **陪跑首次体验**：全程陪同，收集用户反馈并优化。通过品质评估后签署合作协议，正式上线

### 规模化路径

| 阶段 | Host 数量 | 入驻模式 | 品控机制 |
|------|----------|---------|---------|
| Phase 1 · 大同验证（0–6月） | 5–10 | 团队全程深度协作，每 Host 一个主 SKU | 团队人工确认，陪跑首场 |
| Phase 2 · 2–3城市（6–18月） | 20–50 | 结构化表格+远程评审+模板脚本 | Phase 1 Host 参与评审+抽检 |
| Phase 3 · 全国（18月+） | 100+ | 自助入驻→提交审核→认证上线 | 社区评价+算法质量评分 |

Phase 1 Host 成为 Phase 2 的评审人和导师——这是品控机制从人工走向规模化的核心过渡逻辑。

## 八、冷启动计划

### Week 0–2：从最近的圆心出发

初期用户不来自广播式推广，而是从最近的人开始：认识创始人、对大同文化有兴趣、有潜在参与意愿的人群。前5–10单可以完全人工处理（表单+微信确认），不需要等小程序开发完成。

目标不是量，而是验证：**用户愿不愿意付钱？体验过后有没有想讲给别人听的故事？**

### Week 3–8：口碑验证阶段

每次体验后深度收集反馈，聚焦两个问题：高光时刻是什么？愿意推荐给谁？

壁画修复工作坊等稀缺体验有天然传播属性——"我在云冈做了一件事"是一条自发的小红书内容素材。每次高质量体验本身就是获客渠道，边际成本接近零。

### Month 2+：酒店渠道激活

精品民宿和酒店前台无法回答"周边有什么好玩的"——这是一个真实的痛点。「本地」的解决方案：酒店放置体验物料/二维码，按成交给5%佣金。执行门槛极低，是口碑验证过关后的自然扩展。

### 淡旺季应对

大同旅游有明显淡旺季（11月–次年3月）。M4 开始主动 BD 企业团建客户——客单价更高（¥500–800/人），批量订单，可提前确认档期，作为淡季 B2B 收入桥接。

## 九、阶段性路线图

### Phase 1 · M1–M6 · 大同单城验证

**目标：用最小投入验证核心假设，不追求规模**

- **Week 1（最优先，必须早于任何产品开发）**：法务合规确认
  - 代收代付是否需要支付牌照？备用方案是否可行？
  - 提供文化体验预订是否需要旅行社资质？
  - 非执照持有者提供付费体验是否触发导游证要求？
- **Week 1–2**：第一批 Host 接触启动；MVP 方案确认（微信小程序或表单+人工均可）
- **Week 3–4**：首批5笔真实付费订单
- **M1–M4**：执行三引擎爬坡计划，目标 M4 月平台收入 ≥¥5万
- **M3+**：与精品民宿建立酒店渠道合作；开始接触企业团建客户
- **M6 检验节点**：Host 留存率 ≥80%；用户满意度 ≥4.3/5；每5单中 ≥1单来自推荐

### Phase 2 · M7–M18 · 第二城市扩展

目标城市：西安或洛阳（游客基数大、文化资源密度高、尚无同类竞品）

- 输出大同验证的 Host 孵化方法论至新城市，Phase 1 Host 参与新城市评审
- Host 自助发布后台上线；自动结算（T+7）接入
- 入境游模块启动：英文产品描述、首批双语 Host（Airbnb 2022年退出后此空缺至今无人填补）

### Phase 3 · M19+ · 全国化

前提：Phase 2 的城市复制方法论已被验证

- 扩展至成都、南京、泉州、扬州等文化深度二线城市
- 结构化体验元数据库达到可对外提供 API 的规模（对 AI 旅行规划工具有长期商业价值）

## 十、风险与不确定性

我们认为诚实列出风险比回避它们更有意义：

| 风险 | 概率 | 影响 | 应对思路 |
|------|------|------|---------|
| **云冈研究院无法合作**（供给侧） | 未知 | 高 | 这是最核心的单点风险。旗舰锚点能否落地，需要尽快进入正式接触。如无法推进，需重新评估大同单城模型整体 |
| **用户不愿付¥398–498**（需求侧） | 中 | 高 | 从¥298试水；云冈研究院背书如果成立可锚定高价感知；从认识创始人的熟人圈开始最小化验证 |
| **高频SKU上座率达不到预期** | 中 | 高 | M1 是校准月，接受低上座率；核心任务是找到哪个SKU最容易成团，而不是一开始全押 |
| **支付合规阻断**（法务） | 中 | 高 | Week 1 法律咨询必须先行；备用：用户直付 Host+平台事后收佣；或接入连连支付等持牌 MoR |
| **Host 不稳定或流失**（供给侧） | 低 | 高 | Phase 1 人工深度陪跑；协议设计留存激励；不设排他条款（但设内容共有） |
| **旅游淡季现金流**（运营） | 高 | 中 | M4 开始 BD 企业团建，提前锁定淡季收入桥接 |
| **OTC 或资本方进入北方**（竞争） | 低 | 中 | 速度优先：尽快与云冈研究院签正式合作协议，建立先发壁垒 |

**一个需要特别说明的底层不确定性**：目前我们在供给侧的所有"接触方向"，均尚未转化为任何书面协议或正式合作意向。Host 供给是整个模型能跑起来的前提，也是 Phase 1 最需要验证的事情之一。如果在 M1 结束时，5个体验产品中只有2–3个能真正上线，整个爬坡节奏需要重新调整。

## 十一、我们现在在哪里

### 目前真实拥有的

- **对这个市场的理解**：一份相对系统的调研（竞品分析、65条非遗工坊样本、10位用户访谈、单位经济模型），对这件事为什么可行有基于证据的判断
- **产品与运营框架**：即本文档——三引擎收入模型、M1-M4 爬坡计划、Host 孵化逻辑
- **大同本地的初步认知**：创始人有过在大同组织文化体验活动的经历（拾古学社），对本地场域有基本了解，对哪些方向值得优先接触有初步判断——这是认知资产，不是运营资产
- **自有资金**：创始团队有能力自主支持 Phase 1 启动

### 目前还不确定/还没有的

- 核心 Host 的正式确认（5个体验产品能不能上线，完全取决于这件事）
- 法务合规确认（Week 1 的最高优先级）
- 首批5–10笔真实付费订单的验证
- 至少一位深度参与运营的核心成员（有本地生活平台运营或文旅产品经验优先）

### 如果你对这件事感兴趣

我们希望找到对这个方向认真感兴趣的人——不一定是背景完全匹配，而是对"把有价值的在地体验变成可预订、可信任的产品"这件事有真实的判断和投入意愿。

最好的开始方式：你怎么看这个问题？你会在哪里质疑这个模型？哪一部分你认为我们想错了？
