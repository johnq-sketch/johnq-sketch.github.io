01 · 团队

为什么是我们

「本地」的起点不是一个商业判断，而是一段已经发生的历史。拾古学社（Academia Antiqua）在大同和山西的文化遗产场域持续运营深度游学营，已完成多期，积累了真实的付费用户、真实的机构合作关系、以及对"什么样的文化体验让城市外的人愿意付钱"的第一手判断——不是调研推断，而是经历过完整交付、看过用户反应的直接理解。

这意味着：云冈石窟研究院文保部门、东南邑艺术街区、佳游大同——这些不是待确认的意向，而是经过真实合作验证的关系。拾古学社的往期营员，是「本地」平台的第一批天然用户，他们已经信任我们，无需从零建立。

「本地」是这段积累的自然延伸，不是一个全新假设。我们要把已经在小范围跑通的东西，变成一个可预订、可扩展、可持续盈利的产品。

02 · 问题

我们在解决什么

需求侧：旅行者想要"真实"，但找不到入口

越来越多的旅行者不满足于景点打卡——他们想进入真正的日常：当地人去哪里吃早饭、周末做什么、哪条街只有本地人才走。现有渠道给不了这些：OTA 卖门票和标准化行程；内容平台（小红书、抖音）能提供灵感，但没有成交和履约闭环；民宿能住进去，但没有人带你真正进入这座城市的日常。

"不会参加纯观察式的。觉得'做'比'看'更吸引我——要真的动手，或者有真实的对话，才会觉得这趟值了。"

— 访谈用户，25–35岁，互联网从业者

用户访谈明确：参与式体验的心理定价是观察式的3–5倍。当体验绑定了真实专家资质（研究员、修复师、文史学者），受访者愿意支付"景点门票的几倍"。需求是真实的，缺的是可信任的入口。

供给侧：真正的体验提供方，没有预订闭环

这不是大同独有的问题。我们对65个非遗体验工坊（大漆、剪纸、金缮等品类）进行了系统调研，得到一个结论：

市场现状：100% 的成交路径都走这条路

小红书帖子 → 私信询问 → 加微信 → 确认报名 → 转账

没有一家做到"点击即订"。没有退款保障，没有品控机制，出了问题无处投诉。其中点赞最高的案例（6,979赞）背后，用户同样只能通过私信渠道成交——内容热度不等于交易效率。

这套摩擦链路的后果是：有意愿的用户大量流失在"问了没回、回了嫌麻烦"的环节；有内容的工坊获客极度依赖平台流量，没有可持续的预订系统。

旅行者的三道墙

  • 内容平台能种草，无法直接成交履约
  • 支付与信任保障缺失，出问题无处投诉
  • 每次预订耗时极长，平均 10+ 次私信往来

体验提供方的困境

  • 不知道自己的日常有商业价值
  • 获客完全依赖平台流量，无结构化工具
  • 售前沟通成本居高，无法专注内容本身

"中年人并不是不愿意为活动付钱，而是希望看到'研究员''专家''政府支持'这样的字眼。人员和景点深度绑定，专业性是最大的吸引力。"

— 访谈用户，文化遗产旅行爱好者

① 文化事件激活流量——但供给严重滞后。《黑神话：悟空》27个取景地落在山西，大同、云冈成为全国级热搜目的地。游客到了，但除了景点门票，几乎没有更多可以"买"的在地体验。这是一个需求被激活、供给空白的窗口——正是进场时机。

② Z世代旅行行为系统性转变。"在地体验"、"本地人带你玩"在小红书的搜索量连续三年双位数增长；非遗工坊、城市漫步在内容平台上有真实付费转化——65条大漆工坊样本中，头部工坊单帖评论量超200条，其中大部分是真实报名询问。需求不是趋势预测，是已经发生的行为。

③ 政策与地理双重利好。文旅部持续推动"沉浸式体验"与"文化遗产活化"，北方文化遗产城市成为新投资热点。与此同时，现有非遗体验工坊99%集中在成都、上海、武汉等南方城市——大同所在的北方市场，是真实的地理空白。

为什么没人先做

竞品 定位 北方文化遗产覆盖 核心结构性缺陷
OTC 小镇 国内最近竞品，南方慢生活体验 ❌ 无（大理、泉州、杭州、闽南） 产品美学是"自然·治愈·慢生活"，与北方文化遗产的内容密度和叙事语言完全不匹配
携程玩乐 头部OTA体验板块 ⚠️ 有景点门票，无深度非标体验 SKU标准化逻辑，无法承载非标体验；抽佣30%+，Host不愿进
美团玩乐 本地生活体验预订 ⚠️ 以低价团购为主 价格竞争逻辑，无法托举高品质小众体验；定价天花板低
小红书（私域成交） 内容种草 + 私信成交 ✅ 内容覆盖，但无成交闭环 横向平台不做供给侧筛选；正在封堵私域交易，此趋势对「本地」是利好

关键问题一：为什么 OTC 没有北上？

这是投资人必然追问的问题。答案不是"OTC 没想到"，而是有结构性原因：OTC 的产品方法论是「自然·慢生活·治愈系」——农场采摘、花饼制作、Gin 酒工坊——这套叙事语言在云南大理极为自洽，却与大同的壁画修复、西安的器物美学、洛阳的古建叙事完全不匹配。北方文化遗产体验需要更高的内容密度、不同的 Host 孵化逻辑、以及更深的机构合作路径。这是认知壁垒，不是资源壁垒——这正是「本地」的切入点。

关键问题二：小红书/抖音加速布局交易层怎么办？

横向内容平台永远无法解决的问题是：他们不做供给侧筛选和产品化。小红书可以给你一千条"大同值得去"的内容，但无法告诉你哪个向导经过品质认证、无法处理退款、无法给你一个可以信任的陌生人。从65个工坊样本来看，即便内容热度极高（单帖6,979赞），成交路径依然是私信+微信转账——内容平台没有动力去解决这个结构性问题，因为那不是它的商业模型。当大平台建好交易管道，「本地」是他们最需要的内容供给方，而不是竞争对手。

05 · 产品

我们卖什么

好的体验产品的四个判断标准

叙事先行

每个体验必须能回答"我为什么在这里、这件事为什么重要"。文化解释权是最核心的差异化，也是定价权的基础。

主动参与

观看不是体验，动手或真实对话才是。参与式体验的心理定价是观察式的3–5倍——这是访谈数据，不是假设。

在地不可替代

这个体验只能在这里、只能由这个人来完成。稀缺性是定价权和防御壁垒的来源——云冈研究院的合作关系无法被复制。

有东西可以带走

实物带走物是付费意愿的催化剂（"花钱换到了一件东西"）、口碑传播的物理载体（发小红书的道具）、以及情感锚点。大漆工坊市场的核心成功因素之一。

06 · 商业模式

三引擎收入结构

单一抽佣模型的问题：¥298 × 25% × 5人/场 = ¥373/场。要达到月平台收入¥5万，需要约671单——对单城早期文化体验平台，这不现实。核心矛盾不在抽佣比例，而在于每场可捕获价值太低、收入来源过于单一。真正的解法是三台引擎分工协作：

引擎 A · 品牌锚点
云冈旗舰公开场——高客单，不走量

每场4–6人，限额排期，固定时段，带编号带走物。不追求高频次，而是靠它撑起品牌可信度。没有这个锚点，城市漫步就是普通旅游产品；有了它，所有 SKU 都在"我们是做云冈研究院壁画修复体验的平台"这个背书下销售。

单场测算：6人 × ¥498 × 28% = ¥836 平台毛收入 / 月8–16场目标
引擎 B · 上座率主粮
非云冈高频 SKU——真正养活平台的收入来源

城市漫步、东南邑空间叙事、广灵剪纸、食物市集。每场6–12人，工作日+周末均可排期，平台对 Host 的控制权更强，上座率更可预测。云冈的稀缺性决定了它只能做贵不能做多；这些 SKU 的可复制性决定了它们能高频出团，成为收入主干。

城市漫步示例：10人 × ¥198 × 25% = ¥495 / 场 · 月目标40–60场
引擎 C · 高毛利补充
加购与衍生收入——把薄抽佣变厚

编号带走物（壁画颜料临摹卡装框版）、联名讲义/证书、系列票/双场通票。体验本身议价空间有限；加购毛利率显著高于抽佣，是最自然的利润扩展方式。

目标转化率15–20%参与者 · 平台毛利¥50–80/件

M1–M4 收入爬坡（大同单城）

时间节点 月场次 Host 数 平台月收入 核心任务
M1 ~30场 3–4 ~¥1.2万 校准：哪个SKU最容易坐满？哪个Host最稳？
M2 ~48场 5–6 ~¥2.6万 扩张：城市漫步双路线，剪纸正式产品化

两个运营纪律：① Phase 1 每个 Host 只配一个主 SKU，避免脚本和品控同时失控；② 工作日做频次（城市漫步/剪纸），周末做利润（云冈旗舰/空间叙事）。月收入¥5万不是靠某个爆品单打，而是"1个锚点 + 3–4个高频SKU + 加购层"组合驱动。

Host 孵化：帮他们"看见"自己的价值

Phase 1 的核心工作不是招募，而是帮助潜在 Host 发现并描述自己的体验价值，降低他们的启动门槛：

步骤一
深度访谈（1–2次，30–60分钟）

挖掘 Host 最独特的知识、技能、日常场景；确认运营意愿和时间投入上限。明确"这个体验只有你能讲"的核心点在哪里。

步骤二
产品化协作

团队协助撰写产品文案、设计体验流程脚本（开场 → 核心内容 → 高光时刻 → 带走物 → 结尾）、确定定价。Host 专注内容，团队负责产品化包装。

步骤三
陪跑首次体验

团队成员全程参与，收集用户反馈并提供优化建议。通过品质评估后签署合作协议，正式上线排期。

08 · 冷启动与增长

冷启动路径

Week 0–2 · 用现有关系启动
不是广播，是个人通知

向拾古学社往期营员发送定向邀请——这批用户已经信任我们，对大同文化有明确兴趣，且有付费先例。"我们在做一件新事情，你是第一批人。"不追求订单量，追求首批5单的品质和反馈。前5–10笔订单完全可以人工处理，不需要等技术产品就绪。

Week 3–8 · 口碑验证阶段
高光时刻是最好的增长引擎

每次体验后深度收集用户反馈，聚焦两个问题：高光时刻是什么？愿意推荐给谁？壁画修复工作坊等稀缺体验具备自发传播属性——"我在云冈做了一件事"天然适合小红书内容。每次高质量体验本身就是一条可传播的内容，这是边际成本接近零的获客渠道。

Month 2+ · 酒店渠道激活
解决酒店前台答不上来的问题

住客问"周边有什么好玩的"，前台答不上来——这是每家精品酒店都存在的真实痛点。「本地」提供零库存风险解决方案：酒店放置体验物料/二维码，按成交给5%佣金。执行门槛极低（仅需打印物料），为酒店提供即时价值，不要求任何排他承诺。

增长飞轮

飞轮逻辑

优质 Host → 高质量体验 → 用户自发产出内容（小红书/抖音） → 更多订单 → 平台品牌可信度提升 → 吸引更多优质 Host 入驻 → 边际获客成本下降，体验品质持续上升。

这个飞轮的启动成本很低，因为云冈旗舰体验本身是天然的内容素材，不需要额外投放。飞轮能转起来的前提，是第一批5–10次体验的口碑质量过关。

淡旺季现金流管理

大同旅游有明显的淡旺季（11月–次年3月客流显著下降）。应对策略：企业团建作为淡季 B2B 收入桥接。团建客单价更高（¥500–800/人），批量订单，可提前确认档期。M4 开始主动 BD 大同周边企业客户，在旺季结束前完成首批签约，避免现金流断层。
