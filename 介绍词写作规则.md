# 项目介绍词写作规则

## 一、写作总目标

这段项目介绍不是单纯回答"这个项目是什么"，而是要同时回答三个问题：

1. 我为什么会被这个问题吸引
2. 我如何把一种直觉、观察、震动或困惑推进成系统工作
3. 这个项目最终在处理什么更大的问题

始终把文字写成：**一个问题如何进入我，又如何被我做实的过程**。

## 二、标准结构：五步法

### 第一步：用一个具体经验、场景或观察开头

开头的作用不是抒情，而是建立感知入口。必须具体——一个场景、一个瞬间、一次观察、一种不适感、一个反常识现象、一个迟迟放不下的问题。

**禁止**以"我做了一个项目"、"本文主要研究"、"这个项目聚焦"开头。

### 第二步：从经验中抽出一个判断

不能只停在"我感受到什么"，要把经验翻译成一句隐含判断：

- 建筑先塑造身体，再塑造情绪
- 技术不是中性的，它会重构关系
- 产品不是工具，而是一种行为组织方式
- 市场需求背后其实是某种制度空白

这一层决定整段有没有思考深度。

### 第三步：引入一个概念、框架或研究视角，为经验命名

顺序必须是：**先有感觉/观察 → 后有概念命名**，而非先学理论再套现实。理论只起"命名"和"解释"作用，不能压过经验本身。

### 第四步：制造"软转硬"的断点

前面写的是感受、场景、直觉、问题意识，到了这里要突然但自然地落到"我做了什么"。常见句式：

- 正是带着这种感受，我开始了……
- 我没有停留在这种观察本身，而是进一步将它转化为……
- 正因如此，我把这种困惑推进为……
- 最终，我围绕这个问题完成了……

这个转折不能太早（会显得干），也不能太晚（会显得飘）。功能是证明：**我不是只会感受，而是能把感受转化成知识生产、系统研究、产品设计或现实行动。**

### 第五步：用最硬的一句话交代项目内容与更大的问题

不能只写"项目研究了什么"，还要写清楚：

- 项目聚焦什么对象/阶段/场景
- 关注什么机制/矛盾/问题
- 我真正追问的更大问题是什么

从 **对象 → 问题 → 意义**，最后一句必须把项目抬到更大的层面。

## 三、万能公式

```
一个具体场景/经验/观察
→ 我意识到其中隐藏着一个更深的问题
→ 后来通过某个理论/视角/概念，我才更清楚地理解这个问题
→ 正是带着这种问题意识，我把它转化为一个系统项目/研究/产品/实验/分析
→ 这个项目具体聚焦什么，并试图回答一个什么更大的问题
```

## 四、不同类型项目的写法变体

### 研究型项目
适用于：论文、课程paper、田野、历史研究、实验研究。
结构重点：经验入口 → 问题意识 → 研究对象 → 研究问题 → 理论与方法

### 创业/产品项目
"软"的部分可以是：用户痛点、一次失败体验、对市场错配的观察、对某种低效流程的不耐烦。
结构重点：用户场景 → 痛点判断 → 产品化过程 → 验证方式 → 商业与组织含义

### 商业分析/咨询/战略项目
"软"的入口可以是：一个行业悖论、一组互相冲突的数据、一个被忽视的用户行为、一个市场表面繁荣但结构失灵的现象。
结构重点：现象 → 拆解 → 分析框架 → 关键发现 → 决策意义

## 五、四条硬要求

1. **开头必须有画面或抓手** — 先让读者进入一个具体经验、观察或困惑
2. **中间必须有"问题意识升级"** — 不能从现象直接跳到结果，中间必须有一句把问题拔高的话
3. **必须保留"软转硬"** — 让读者感觉到：我有感受力，但我并不止于感受，我会把感受推进成研究、产品、论证或行动
4. **结尾必须指向一个"大问题"** — 最后一句不能只介绍项目内容，要把项目抬到更大的层面

## 六、语气要求

- 语言克制、清楚、有文气，但不空泛
- 不堆砌抽象词
- 不写流水账
- 不写纯学术摘要
- 不为"高级"而失去信息密度
- 既保留人的质感，也体现分析能力和硬实力
- 项目更硬则"软"的部分缩短；项目更抽象则"软"的部分写得更扎实

## 七、输入格式

根据以下信息写作，缺项则根据已有信息补出完整结构：

- 项目类型：
- 项目名称：
- 我最初为什么会开始这个项目：
- 一个最能代表这个项目的场景/观察/问题：
- 这个项目主要研究或解决什么：
- 我具体做了什么：
- 项目的核心发现/判断：
- 这个项目背后更大的问题是什么：
- 希望整体风格更偏：文气/克制/研究型/申请文书型/创业叙事型

## 八、输出要求

输出一段完整文字，不是提纲。必要时可分两段，整体必须连贯自然。优先保证：

- 入口具体
- 判断清楚
- 转折自然
- 项目内容交代明确
- 最后有更大的问题意识

## 九、范例：东大寺项目

> 奈良的寺庙里，檐角压得极低，光从格子窗漏进来，人还没开口，就已经先安静了。那并不单纯出于虔诚，而是空间先一步塑造了你的身体，也替你安排好了将要进入的情绪。很多年后读到安藤忠雄的采访，我才明白，少年时的我在一次次出入宗教建筑时，身体先于语言所感受到的，究竟是什么。安藤所说的"自然"，并不只是花木风雨意义上的自然，而更接近一种崇高的意向，一种几乎带有神性的存在。建筑之所以重要，恰恰在于它替这种无形之物划出边界，让光影、尺度与材料把那些无法直接说出的情绪一层层压到人身上。
>
> 于是，当我真正站到东大寺面前时，才第一次如此清楚地感到：信仰并不一定是无形的，它完全可以借由空间、物质、秩序与体量，被一点点建造出来。正是带着这种感受，我把最初那种近乎本能的震动，转化为对东大寺的系统研究，并于 2021 年发表了 Tōdaiji: A State Appropriation of Buddhism。文章主要聚焦圣武天皇时期，关注佛教如何在这一阶段被纳入国家治理与政治整合的框架之中，以及东大寺如何通过其制度位置、营建过程与建筑形态，成为这种国家对佛教的吸纳与征用最集中的体现。对我而言，这篇文章并不只是对一座寺院的历史考察，更是在追问：一种原本指向超越性的信仰，如何被转化为一种可以服务于现实秩序、并被权力结构具体调动的力量。
